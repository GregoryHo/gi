# Subagents roadmap

## M1 — Foreground read-only facade

Ship one bounded parallel `subagent` tool over Agent Workers protocol v1 with built-in read-only agent definitions and stable foreground results.

## Later candidates

- Agent-definition discovery from Markdown, only after a safe precedence and validation design.
- Explicit narrow context inheritance, only after privacy and prompt-size policy.
- Background or persistent sessions, only after lifecycle and ownership design.
- Write-capable delegation, only with an explicit authority and confirmation model.

## Separate future product

Shared tasks, teammate messaging, assignment, membership, retries, synthesis, and other coordination semantics belong to a future Agent Teams package, not Subagents.
