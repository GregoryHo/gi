# Web Search versioned planning

Use this directory for product iterations after the initial `0.1.0` MVP planning era.

Expected structure:

```text
versions/<semver>/
├── index.md
├── milestones.md
├── log.md
└── m<N>-<topic>.md or YYYY-MM-DD-<topic>.md
```

Root-level `milestones.md` and milestone plans may be used for the initial MVP. After `0.1.0` ships, future minor versions should be planned under `versions/<semver>/` before implementation starts.
