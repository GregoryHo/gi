# M5 — 1.0.0 sealing

## Status

Planned.

## SPEC

### Goal

Declare Web Search 1.0.0 stable after public GitHub clone usability, metadata/license, diagnostics, and release-candidate verification are complete.

### Scope

- Bump `packages/pi-extension-web-search/package.json` version to `1.0.0`.
- Update `packages/pi-extension-web-search/CHANGELOG.md` with a dated `1.0.0` section.
- Update `docs/release-policy.md` package table with `1.0.0`.
- Update `docs/pi-extension-web-search/index.md`:
  - stable version `1.0.0`.
  - active planning version `None` unless the next version has started.
  - summarize 1.0.0 public GitHub readiness.
- Update `docs/pi-extension-web-search/archive.md` with 1.0.0 docs links.
- Update `docs/pi-extension-web-search/log.md` with release/sealing notes and verification evidence.
- Mark `versions/1.0.0/index.md` status `Sealed`.
- Ensure all 1.0.0 milestones are `Done`.
- Optionally create a package-scoped tag only if explicitly requested:

```bash
git tag pi-extension-web-search/v1.0.0
```

### Non-goals

- Bare semver tags such as `v1.0.0` or `1.0.0`.
- npm publication.
- New runtime features.
- Direct root `pi install git:...` support.

### Design notes

- Sealing should be mostly documentation and version metadata.
- Do not declare 1.0.0 stable until M4 clean clone evidence exists.
- If a tag is created, use only the package-scoped tag required by `docs/release-policy.md`.

### Expected files

- `packages/pi-extension-web-search/package.json`
- `packages/pi-extension-web-search/CHANGELOG.md`
- `docs/release-policy.md`
- `docs/pi-extension-web-search/index.md`
- `docs/pi-extension-web-search/archive.md`
- `docs/pi-extension-web-search/log.md`
- `docs/pi-extension-web-search/versions/1.0.0/index.md`
- `docs/pi-extension-web-search/versions/1.0.0/milestones.md`
- this milestone plan completion notes

## AC

- Package version is `1.0.0`.
- Changelog has a dated `1.0.0` entry.
- Docs identify 1.0.0 as the stable version and no active planning version remains.
- Archive links all completed 1.0.0 plans.
- All verification commands pass.
- Clean clone smoke evidence is recorded from M4.
- If tagged, tag is `pi-extension-web-search/v1.0.0`, not a bare semver tag.

```bash

```
