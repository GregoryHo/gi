# Web Search 0.5.1 planning

## Status

Sealed.

## Goal

Improve manual smoke-test and user observability by surfacing fetched-content retrieval metadata in visible tool output.

## Milestones

- M1 — `m1-visible-retrieval-metadata.md`: completed visible `responseId`, full character count, offsets, and next offsets in `fetch_content` / `get_search_content` text output.

## Non-goals

- Changing storage semantics.
- Persistent fetched-content storage.
- HTML extraction changes.
- Search provider changes.
- New network behavior.
